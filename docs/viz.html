Pregunta 1 – Distribución por grupo

Dataset

Importamos el dataset de penguins desde la fuente proporcionada, realizamos una limpieza de datos eliminando valores nulos y corrigiendo el error de registro en la columna de sexo. Esto nos permite trabajar con un conjunto de datos limpio y listo para el análisis.:

import pandas as pd
import altair as alt

url = "https://raw.githubusercontent.com/erickedu85/dataset/master/penguins.csv"
df = pd.read_csv(url)

# 2. Limpieza de datos
df = df.dropna()            # Eliminar nulos
df = df[df['sex'] != '.']   # Eliminar error de registro "." en la columna sex

¿Cómo varía la masa corporal (body_mass_g) entre las diferentes especies e islas de los pingüinos?

¿Qué tipo de gráfico usarías para comparar la distribución de una variable cuantitativa entre varios grupos definidos por dos variables categóricas?

Se puede utilizar diferentes gráficos para presentar los datos, desde un gráfico tipo Heatmap de conteo, un boxplot, entre otros, en esta ocasión se uso el boxplot por la facilidad de interpretar los datos

¿Cómo facilitarías la comparación entre subgrupos sin sobrecargar un solo gráfico?

Se puede utilizar un gráfico tipo mark_circle para ello, ya que permite identificar la masa corporal, así como la especie y el sexo de los pingüinos.

Implementa tu elección y analiza visualmente qué patrones se observan.

Se puede observar que la variable “body_mass_g” cambia de acuerdo con la especie y el género de los pingüinos, en general los machos tienen un índice de masa corporal mayor a las empras, además se observa que los pingüinos de la especie Gentoo presentan valores mayores.

alt.Chart(df).mark_bar().encode(
    x=alt.X('body_mass_g:Q', bin=alt.Bin(maxbins=30), title='Masa Corporal (g)'),
    y=alt.Y('count()', title='Cantidad de Pingüinos'),
    color='species:N'
).properties(
    width=600,
    height=400,
    title='Distribución de la Masa Corporal por Especie'
)

Pregunta 2 – Ubicación de especies

¿Qué especies de pingüinos se encuentran en cada isla? ¿Alguna especie está presente solo en una isla?

¿Qué técnica visual te permitiría mostrar esta relación de forma compacta y clara?

Para observar las especies ubicadas en las islas se puede utilizar un Heatmap de conteo, con lo cual se observa las especies en cada isla, así como las islas que tienen mas de una especie

Aplica una visualización que permita identificar asociaciones entre categorías.

Como se menciona previamente el Heatmap de conteo permitirá analizar las variables categóricas y su frecuencia de ocurrencia, lo que permite identificar las correlaciones entre especies e islas.

Interpreta el resultado.

Para este punto se observa que en la especie Adelie habita en las tres islas, con una población ligeramente mayor en la isla Dream, por otro lado la especie Chinstrap habita únicamente en la isla Dream, finalmente la especie Gentoo habita en la isla Biscoe, pero tiene una población mayor a las otras especies.

# Usamos 'df' que es tu variable con los datos
alt.Chart(df).mark_rect().encode(
    x=alt.X('island:N', title='Isla'),
    y=alt.Y('species:N', title='Especie'),
    color=alt.Color('count():Q', title='Cantidad', scale=alt.Scale(scheme='greens'))
).properties(
    title='Presencia de Especies de Pingüinos por Isla',
    width=300,
    height=200
)

Pregunta 3 – Diferencias entre sexos

¿Existen diferencias notables entre machos y hembras en alguna característica cuantitativa?

Elige una variable cuantitativa adecuada (como culmen_length_mm, culmen_depth_mm, etc.)

Para este caso se elige la variable bpdy_mass_g, se identifica que los machos tienen un peso mayor que las hembras en todas las especies

Justifica el gráfico que usarías para comparar distribuciones entre dos categorías.

El boxplot permite una comparación entre categorías al representar estadísticos de la distribución (mediana, cuartiles y valores extremos), lo que facilita la identificación de diferencias estructurales entre grupos.

alt.Chart(df).mark_boxplot(extent='min-max').encode(
    x=alt.X('sex:N', title='Sexo'),
    y=alt.Y('body_mass_g:Q', title='Masa Corporal (g)'),
    color='sex:N',
    column=alt.Column('species:N', title='Especie') # Separar por especie es vital aquí
).properties(
    width=150,
    title='Comparación de Masa Corporal entre Sexos por Especie'
)
